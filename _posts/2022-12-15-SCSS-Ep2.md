## __주석__
---

```scss
.container {
	h1 {
			color: red;
			/* background-color: blue; */
			// font-size: 60px;
		}
}
```

- `/* */`
  
  해당 주석은 scss에서 css로 변환(컴파일)했을 때, 주석처리가 남아있다.  
  **⇒ 코드가 남아 있다.**
    
- `//`
  
  해당 주석은 scss에서 css로 변환했을 때, 코드가 나오지 않는 (존재하지 않는)다.  
  **⇒ 코드가 남아 있지 않다.**
