## __예약어__
---
예약어는 **특별한 의미**를 가지고 있어, 변수나 함수 이름 등으로 사용할 수 없는 단어이다.
- **Reserved Word**
  ```js
  let this = 'Hello'; // SyntaxError
  let if = 123; // SyntaxError
  let break = true; // SyntaxError
  ```
